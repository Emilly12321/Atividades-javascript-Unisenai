ATIVIDADE FINAL

Este site foi desenvolvido com a finalidade de ser um repositório de atividades de programação utilizando HTML, CSS e JavaScript. Nele, os conteúdos estão organizados por estruturas e temas, permitindo que o usuário selecione uma opção e visualize exemplos, atividades propostas e suas respectivas soluções. Ao escolher uma atividade, é exibido o enunciado junto com a possibilidade de acessar o código e também executá-lo diretamente no site. Dessa forma, este repositório serve tanto como material de apoio para consultas futuras quanto como um recurso para auxiliar no processo de aprendizagem e prática da programação.
