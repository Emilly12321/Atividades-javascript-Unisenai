UNIDIMENSIONAL

Um array unidimensional é uma estrutura de dados que armazena uma coleção de elementos do mesmo tipo organizados em uma única linha. Cada elemento é acessado por um índice numérico que geralmente começa em zero. Arrays desse tipo são úteis para representar listas simples, como nomes de alunos, notas ou números sorteados. Eles permitem armazenar vários valores em uma única variável, facilitando o uso de laços de repetição para leitura, exibição ou processamento dos dados.
