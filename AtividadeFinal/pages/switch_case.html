Switch..Case

A estrutura switch-case é uma forma de controle condicional utilizada em linguagens de programação como JavaScript para simplificar a verificação de múltiplas condições baseadas no valor de uma única variável. Em vez de utilizar várias instruções if-else, o switch permite comparar o valor de uma expressão com diferentes casos (case) e executar um bloco de código correspondente. Cada case representa um possível valor, e a execução é interrompida com a instrução break para evitar que os demais casos sejam avaliados. É uma estrutura útil quando se precisa tomar decisões baseadas em valores fixos ou faixas específicas, tornando o código mais organizado e legível.
