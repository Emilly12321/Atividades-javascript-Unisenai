WHILE

A estrutura while é uma forma de controle de repetição usada em linguagens de programação como JavaScript para executar repetidamente um bloco de código enquanto uma condição lógica for verdadeira. Antes de cada iteração, a condição é avaliada; se for verdadeira, o código dentro do loop é executado, caso contrário, o laço é encerrado. Essa estrutura é útil quando o número de repetições não é conhecido previamente, permitindo que o programa continue repetindo enquanto um critério específico for atendido, tornando o código eficiente e flexível para diversas situações.
