DO..WHILE

A estrutura do...while é um tipo de loop utilizado em linguagens como JavaScript que executa um bloco de código pelo menos uma vez antes de verificar uma condição lógica no final de cada iteração. Diferente do while, o do...while garante que o código dentro do laço seja executado uma vez, mesmo que a condição já seja falsa no início. Esse comportamento é útil quando se deseja garantir a execução inicial do bloco e continuar repetindo enquanto a condição permanecer verdadeira, proporcionando controle preciso sobre ciclos de repetição no programa.
